# Django opinion system

## Usage
Clone the repository 

```bash
git clone https://github.com/nowickiszymon/Opinion-System.git
```

## Instalation
```pip 
pip install req.txt
```

## Run
```bash
python manage.py runserver
```

## API Points
GET all opinions
```bash
localhost:8000/api/all
```

POST request
```bash 
localhost:8000/api/create
